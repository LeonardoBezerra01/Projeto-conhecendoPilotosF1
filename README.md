## Conhecendo Pilotos Históricos da Fórmula 1

**Descrição:**

Este projeto web é um diretório simples que permite aos usuários pesquisar por pilotos históricos da Fórmula 1. Ao digitar o nome de um piloto no campo de pesquisa, a aplicação retorna informações relevantes como nome, descrição e um link para mais informações.

**Funcionalidades:**

* **Pesquisa:** Permite pesquisar por pilotos usando palavras-chave.
* **Resultados Dinâmicos:** Exibe os resultados da pesquisa de forma clara e concisa.
* **Links Externos:** Direciona o usuário para fontes externas para obter mais informações sobre o piloto.

**Tecnologias Utilizadas:**

* **HTML:** Estrutura da página.
* **CSS:** Estilização da página.
* **JavaScript:** Lógica da aplicação, incluindo a pesquisa e a manipulação do DOM.
